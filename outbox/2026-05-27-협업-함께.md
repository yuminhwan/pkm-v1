# 협업을 어떻게 하면 잘 할 수 있을 지 생각해보고 적용해본다

> **🧭 첫 줄 주장 (한 문장으로, 본인 손으로 다듬어 채워주세요):**
> 협업을 LLM을 통해서 어떻게 하면 잘 할 수 있을 지 생각해보고 적용해본다

프로젝트가 진행되면서 팀원들과 소통하는 기회가 많이 줄어들고 공유하는 시간이 많이 줄어든 것 같다. 

혼자 생각, 일하고 작업하고 보고하는 식으로 일을 하고 있는데 이게 일하는 방식 및 협업의 올바른 방향은 아니라고 생각이 든다.

이를 '함께 자라기' 책을 통해서 어떤 식으로 협업을 진행하고 소통해야 하는지 조금 알아보고자 한다.

구글이 밝힌 탁월한 팀의 비밀 단락을 봤을 때 탁월한 팀에서는 심리적 안정감이 있고 이는 팀 토론 등 특별히 고안된 활동을 통해서 이루어진다고 한다. 

특별히 고안된 활동이란 무엇일까? 팀 회고가 있을 수 있을 것 같고 주간 논의나 데일리 스크럼도 될 수 있을 것 같다. 

바로 팀 활동으로 진행하면 좋겠지만 일단 내가 진행해보고 어떤 점이 좋았는지 아닌지를 판단해볼 수 있을 것 같다. 

혼자 내가 한번 진행해보고 이를 팀에도 도입해볼 수 있지 않을까 싶다. 

처음에 LLM이 회고한 글에 대해서 평가를 하고자 했지만 이는 LLM을 활용해서 협업을 잘 이루어지게 하는 건 아닌 것 같아서 다른 방향을 제시해달라고 했고 회고 내용을 바탕으로 LLM이 코칭 관점에서 평가하도록 변경해보았다.
