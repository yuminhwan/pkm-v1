# .inbox/

들어오는 모든 raw 자료가 쌓이는 곳. 점(`.`) prefix는 *사람이 정리할 자리가 아니라 그대로 던지는 자리*라는 뜻 

## 들어오는 것 예시

- 스크린샷
- 링크 (URL만이라도 OK)
- 회의 녹취록
- 떠오른 생각 한 줄
- AI에게 던진 프롬프트 + 그 답변 (이것도 raw 자료라 여기 던진다 — 별도 폴더 X)

## 규칙

- **가공하지 않는다.** raw 그대로.
- 파일명은 대충 지어도 된다. 어차피 소화해서 `outbox/`로 옮길 때 새 이름이 붙음.
- 너무 오래 쌓이면 그게 신호 — 소화 못 하고 있다는 뜻이니 한 번 비운다.
- AI 답변을 `outbox/`로 옮길 땐 **라인 단위로만** 가공해서 옮긴다. 단락 통째 복붙 X — 통째로 흡수하면 인출 학습이 안 일어난다.
